Sobre mim

Me chamo Lucas, possuo 25 anos e trabalho na área de tecnologia da informação a 1 ano e 5 meses. trabalhei utilizando ferramentas como SQL Server, postgreSQL e mongoDB, python para visualização e análise de dados, criação de dashboards utilizando Power BI. Atualmente estou intercalando minhas habilidades na área de dados com a área front-end, com HTML, CSS, JavaScript e React.

Trabalhei como Administrador de Banco de Dados na empresa Datamétrica, prestando meus serviços para a caixa econômica federal, em seussistemas para empresas de telecomunicações e telemarketing, cuidei da administração, monitoração, criação, manutenção de jobs e procedures, realizei o levantamento e apuramento de dados utilizando o SQL Server. Elaborava planejamentos e instruí outros administradores de banco de dados, realizando a monitoração e observação deles.

Trabalhei como analista de banco de de dados na equipe de engenharia de dados da Insole. Realizei a análise, tratamento dos dados e soluções a partir de ferramentas de ETL onde uso como ferramentas de trabalhos o mongoDB, Python para visualização de dados e caso necessário a realização de dashboards utilizando o Power BI.
